# AdMaster Pro — Product Roadmap

> Last updated: March 15, 2026

---

## Vision

AdMaster Pro is an AI-powered ad management platform that lets small business owners create, manage, and optimize their advertising across Google Ads, Facebook Ads, Amazon Ads, and Shopify — all from a single intelligent dashboard. The AI handles research, creation, compliance, optimization, and reporting so the user can focus on running their business.

---

## Phase 1: AI Ad Creation Mastery (Current Sprint)

**Goal:** Let AI fully handle ad creation across all Google Ads formats using best practices to avoid policy violations and bans.

### 1.1 — Seamless Onboarding (✅ Building Now)
- [x] Google-style one-step-at-a-time onboarding flow
- [x] Sign up / sign in with Google
- [x] Enter business name + website → auto-scan
- [x] Connect Google Ads account (pull existing ads)
- [x] Connect Shopify store (optional)
- [x] Upload brand assets (logos, product photos)
- [x] AI analyzes existing ads → generates report with improvements
- [x] AI auto-generates optimized ads based on research
- [x] User approves ads before going live
- [x] No pricing/billing during onboarding — everything is free

### 1.2 — AI Ad Creation Engine
- [ ] **Search Ads**: Headlines (30 chars), Descriptions (90 chars), Display URLs, Sitelinks, Callouts, Structured Snippets
- [ ] **Responsive Search Ads (RSA)**: Up to 15 headlines + 4 descriptions, AI picks best combos
- [ ] **Shopping Ads**: Product titles, descriptions, images, pricing optimized for Google Merchant Center
- [ ] **Display Ads**: Responsive display ads with AI-generated text overlays, proper dimensions (300x250, 728x90, 160x600, etc.)
- [ ] **Performance Max (PMax)**: Asset groups with headlines, descriptions, images, videos, audience signals
- [ ] **Video Ads (YouTube)**: Script generation, thumbnail suggestions, call-to-action overlays
- [ ] **App Ads**: Text, image, and video assets for app install campaigns
- [ ] **Demand Gen Ads**: Carousel and single-image ads for Discover, Gmail, YouTube
- [ ] **Call-Only Ads**: Optimized for mobile click-to-call

## Data Isolation & Security

- Every user's data is strictly isolated by `userId` and `businessId`
- AI never has access to other users' ad data, strategies, or business information
- All Google Ads API calls use the individual user's OAuth credentials
- Shopify data is scoped to the user's connected store only
- Chat context is built from the active user's knowledge base only
- No cross-account data leakage in AI prompts or responses

---

## Architecture Notes

- **Multi-tenant by design**: All models include `userId` foreign key
- **Per-business isolation**: Most models also include `businessId`
- **OAuth per integration**: Separate refresh tokens for Google Ads, Shopify, Facebook, Amazon
- **AI context building**: Chat system loads only the active business's KB items (capped at 8000 chars)
- **Approval workflow**: All AI-generated ads start as drafts → user reviews → publishes
